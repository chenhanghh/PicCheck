##### 简要描述

- 手机号+密码验证码登录接口。

##### 请求URL
- ` http://localhost/users/api/login/sms/ `
  
##### 请求方式
- POST 

##### 参数

| 参数名         |必选| 类型     | 说明  |
|:------------|:---|:-------|-----|
| phonenumber |是  | String | 手机号 |
| captcha     |是  | int    | 验证码 |

##### 返回示例 

``` 
  {
    "code": 200,
    "msg": "用户登录成功！"
  }
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |状态码  |
|msg |String   |响应信息  |

##### 备注 

- 更多返回错误代码请看首页的错误代码描述
