##### 简要描述

- 文件重命名

##### 请求URL
- ` http://localhost/projects/api/renfile/ `
  
##### 请求方式
- POST 

##### 参数

| 参数名       |必选| 类型     | 说明       |
|:----------|:---|:-------|----------|
| file_id   |是  | int    | 项目id     |
| folder_id |是  | int    | 项目所属文件夹id |
| new_title |是  | String | 文件新名字    |

##### 返回示例 

``` 
  {
    "code": 200,
    "msg": "文件重命名成功！"
  }
```

##### 返回参数说明 

|参数名| 类型     | 说明    |
|:-----  |:-------|-------|
|code | int    | 状态码   |
|msg | String | 响应信息  |


##### 备注 

- 更多返回错误代码请看首页的错误代码描述
