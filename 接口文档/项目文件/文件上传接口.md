##### 简要描述

- 文件上传

##### 请求URL
- ` http://localhost/projects/upload/ `
  
##### 请求方式
- POST 

##### 参数

| 参数名        |必选| 类型   | 说明      |
|:-----------|:---|:-----|---------|
| user_id    |是  | int  | 上传者id   |
| project_id |是  | int  | 上传项目id  |
| folder_id  |是  | int  | 上传文件夹id |
| file       |是  | FILE | 上传文件    |

##### 返回示例 

``` 
  {
    "code": 200,
    "msg": "文件上传功！"
  }
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |状态码  |
|msg |String   |响应信息  |
