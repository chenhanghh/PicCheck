##### 简要描述

- 新建文件夹

##### 请求URL
- ` http://localhost/projects/api/addfolder/ `
  
##### 请求方式
- POST 

##### 参数

| 参数名      |必选| 类型     | 说明    |
|:---------|:---|:-------|-------|
| title    |是  | String | 文件夹名称 |
| project_id |是  | int    | 项目id  |

##### 返回示例 

``` 
  {
    "code": 200,
    "msg": "文件夹新建成功！"
  }
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |状态码  |
|msg |String   |响应信息  |


##### 备注 

- 更多返回错误代码请看首页的错误代码描述
