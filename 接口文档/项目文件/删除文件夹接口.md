##### 简要描述

- 批量删除文件夹

##### 请求URL
- ` http://localhost/projects/api/delfolder/ `
  
##### 请求方式
- POST 

##### 参数

| 参数名     |必选| 类型   | 说明      |
|:--------|:---|:-----|---------|
| folder_ids |是  | List | 多个文件夹id |
- folder_ids示例：folder_ids: [31, 32]

##### 返回示例 

``` 
  {
    "code": 200,
    "msg": "文件夹删除成功！"
  }
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |状态码  |
|msg |String   |响应信息  |

- 更多返回错误代码请看首页的错误代码描述
