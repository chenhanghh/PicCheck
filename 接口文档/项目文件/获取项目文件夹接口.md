##### 简要描述

- 获取项目文件夹目录

##### 请求URL
- ` http://localhost/projects/api/projectinfo/ `
  
##### 请求方式
- POST 

##### 参数

| 参数名        |必选| 类型     | 说明     |
|:-----------|:---|:-------|--------|
| project_id |是  | int    | 项目id   |

##### 返回示例 

``` 
  {
    "code": 200,
    "msg": "项目获取成功！"
    "folderslist": [
        {
            "id": 32,
            "create_date": "2023-09-17T06:10:26.200Z",
            "file_count": 1
        }
    ]
  }
```

##### 返回参数说明 

|参数名| 类型     | 说明    |
|:-----  |:-------|-------|
|code | int    | 状态码   |
|msg | String | 响应信息  |
|folderslist | Json   | 文件夹信息 |
