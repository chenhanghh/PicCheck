##### 简要描述

- 文件上传

##### 请求URL
- ` http://localhost/camera/api/upload/ `
  
##### 请求方式
- POST 

##### 参数

| 参数名     |必选| 类型   | 说明      |
|:--------|:---|:-----|---------|
| file    |是  | FILE | 上传文件    |

##### 返回示例 

``` 
  {
    'id': 7, 
    'file': 'http://localhost/upload/uploads/users/20231007/crack_1_jl0Vz9g.jpg', 
    'create_date': '2023-10-07T17:01:14.716838+08:00'
  }
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|msg |String   |响应信息  |
